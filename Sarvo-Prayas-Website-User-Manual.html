2. How to Add / Edit a Post

a. Create New Post

  • Click on "Posts" in side bar in admin dashboard
  • Click the New Post button (top right)
  • Write the Title: Click the large text at the top and type your headline.
  • Write the Content: Click below the title and compose your story.
  • Tip: Press the + icon on a new blank line to insert images, videos, dividers, or quotes!
  • Add a Feature Image: Click the "Add feature image" box at the top of the post to upload the main photo of your beneficiaries or event.
  • When done, click Publish (top right).
  • Publish: When you are ready, click the Publish button in the top right corner.

Important: Always add a Tag before publishing from sidebar.

Important: Only upload images in WebP format. Do not upload JPG or PNG files.

b. How to Edit an Existing Post

  • Click Posts in the left sidebar
  • Find the post you want to edit
  • Click on it to open
  • Make your changes
  • Click Update (top right) to save

3. How to Add a New Ongoing or Past Project

  • Click Posts in the left sidebar
  • Click New Post
  • Write the project name as the Title
  • For an ongoing project: write the full description as the content and add a feature image (WebP format)
  • For a past project: only the project name (Title) is needed. Past projects show as clean name-only cards, so a description and image are not required
  • Click Post Settings / Side bar Icon
  • Scroll down to Tags and add: ongoing-project for a current project, or past-project for a completed one
  • Click Publish

The project will automatically appear on the Ongoing Projects or Past Projects page. The Ongoing Projects page shows 6 per page and the Past Projects page shows 9 per page; when there are more, page numbers appear at the bottom automatically.

7. How to Edit the Homepage (Section by Section)

a. The Hero Slider (Top Background Animation)

  • Create ONE New Post (only one post for the slider).
  • Title: Anything you like, for example "Hero Slider" (it is not shown on the site).
  • In the content, add several images one after another. These become the rotating full-screen backgrounds.
  • Open the Post Settings and add the tag hero-slider.
  • Publish it! The images rotate automatically, one every 5 seconds.
[ Insert screenshot here: Home page hero with background image ]

b. The Two Home Cards (Child Protection & Livelihood)

  • For the Child Protection card: create a post, write the title and short text, add a feature image, and tag it home-focus-child.
  • For the Livelihood card: create a post, write the title and short text, add a feature image, and tag it home-focus-livelihood.
  • Use only one post for each tag.
[ Insert screenshot here: "Two causes. One community." cards on home page ]

c. The Timeline (Milestones)

  • Create ONE post and tag it milestone.
  • In the content, add a bulleted list. Start each bullet with the year in bold, then the description.
  • This same timeline shows on both the Home page and the About page.
[ Insert screenshot here: Timeline section ]

d. The Partner Strip

  • Create ONE post and tag it partner.
  • In the content, list each partner name as a bullet.
  • The names scroll across the home page automatically.
[ Insert screenshot here: Scrolling partner strip ]

e. The Impact Stats (numbers band)

  • Create ONE post and tag it stats.
  • In the content, add a bulleted list. For each stat, make the number bold first, then a space, then the label. For example: 22+ Years of Service.
  • To make the number bold, select it and press Ctrl and B together.
  • This same stats band shows on both the Home page and the About page, so editing this one post updates both places at once.
  • To update a number later, just open this post, change the number, and click Update.
[ Insert screenshot here: Impact stats band (numbers) ]

8. How to Edit the About Page

Each block on the About page is controlled by its own post. Create one post for each, with the tag shown below.

About page block Tag to use What to add
Background / Story about-story Title, paragraphs, feature image
Our Mission card about-mission Title, short text
Our Vision card about-vision Title, short text
Where We Work about-where Title, paragraphs, feature image

The post title becomes the heading and the content becomes the text. Use only one post per tag.

10. How to Update Contact, Bank and Legal Details

This information appears in many places, so it is set in one place. Change it once and it updates everywhere.

  • In the sidebar click Settings, then Design & branding
  • Click Customize, then open the theme settings (Sitewide)
  • Update any of the fields below, then click Save
Field Controls
Hero title The big headline on the home page
Brand tagline The small line under the logo
Contact phone 1 / 2 Phone numbers in footer and contact page
Contact email Email address shown across the site
Office address 1 / 2 Head office and branch office addresses
Society / 12A / 80G / PAN Legal registration numbers
Bank details Account details on the Donate page (single bank account)

11. How to Update Navigation Menu

  • Click Settings in the sidebar
  • Click Navigation
  • To add a new link: Type the label and URL in the empty fields at the bottom
  • To edit an existing link: Click on the label or URL and type the new value
  • To remove a link: Click the X button next to it
  • To reorder: Drag and drop the links
  • Click Save when done

13. The Language Button (English / Hindi)

The website has an EN and a हिं button at the top right. Visitors click it to read the site in Hindi. It uses automatic translation and works on the live website. You do not need to maintain it.

14. How to Use Tags – Quick Reference

Tags control where content appears on the website. Always use the correct tag, in small letters with hyphens.

Tag Use For
ongoing-project Current project posts
past-project Completed project posts
gallery Photo posts
video Video posts
event Event posts
partner One post listing all partners
stats One post for the impact numbers band (Home and About)
milestone One post for the timeline
hero-slider One post with the home background images
home-focus-child Child Protection card on home
home-focus-livelihood Livelihood card on home
about-story About page story block
about-mission About page mission card
about-vision About page vision card
about-where About page "Where We Work" block

Important: Never delete these tags. If a wrong tag is used, the post will not appear in the correct section.

15. Guidelines to Follow

  • Images must be WebP format only – do not upload JPG or PNG
  • Always add the correct Tag before publishing a post
  • Always add a Feature Image to project, gallery and video posts – this is what shows on the cards
  • For the tags marked "one post" (partner, stats, milestone, hero-slider, home- and about- ones), edit the existing post instead of making a new one
  • Do not delete any pages – only edit the content inside them
  • Do not change the page slugs (URLs) listed in section 9
  • If you do not see your change, refresh the page with Ctrl, Shift and R together
  • Save before closing – always click Update or Publish before leaving
