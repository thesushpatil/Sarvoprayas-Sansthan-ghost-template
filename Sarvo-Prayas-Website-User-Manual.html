Sarvo Prayas Sansthan – Ghost Manual

1. How to Login to Ghost Admin

  • Open your browser and go to yourdomain.com/ghost
  • Enter the login credentials (Email and Password)
  • Admin Dashboard will appear
[ Screenshot: Ghost login screen ]

2. How to Add / Edit a Post

a. Create New Post

  • Click on "Posts" in the left sidebar
  • Click the New Post button (top right)
  • Write the Title: Click the large text at the top and type your headline
  • Write the Content: Click below the title and type your text
  • Tip: Press the + icon on a blank line to insert images, videos, dividers, or quotes
  • Add a Feature Image: Click "Add feature image" at the top of the post to upload the main photo
  • When done, click Publish (top right)

Important: Always add a Tag before publishing from the sidebar settings.

Important: Only upload images in WebP format. Do not upload JPG or PNG files.

[ Screenshot: New Post editor ]

b. How to Edit an Existing Post

  • Click Posts in the left sidebar
  • Find the post you want to edit
  • Click on it to open
  • Make your changes
  • Click Update (top right) to save

3. How to Add a New Ongoing or Past Project

  • Click Posts in the left sidebar
  • Click New Post
  • Write the project name as the Title
  • For an ongoing project: write the full description as the content and add a feature image (WebP format)
  • For a past project: only the project name (Title) is needed. Past projects show as clean name-only cards, no image or description required
  • Click the sidebar settings icon
  • Scroll down to Tags and add: ongoing-project for a current project, or past-project for a completed one
  • Click Publish

The project will automatically appear on the correct page. Ongoing Projects shows 6 per page and Past Projects shows 9 per page. Page numbers appear at the bottom automatically when there are more.

[ Screenshot: Projects page ]

4. How to Add a Photo to the Gallery

  • Click Posts in the left sidebar
  • Click New Post
  • Write a short caption as the Title
  • Add a feature image (the photo in WebP format)
  • Add Tag: gallery
  • Click Publish

The photo will appear on the Photos page. Shows 6 per page.

5. How to Add a Video

  • Click Posts in the left sidebar
  • Click New Post
  • Write the video name as the Title
  • In the content, paste the YouTube link on its own line (it turns into a player)
  • Add a feature image (a thumbnail picture in WebP format)
  • Add Tag: video
  • Click Publish

6. How to Add an Event

  • Click Posts in the left sidebar
  • Click New Post
  • Write the event name as the Title
  • Write the event details as the content
  • The Publish Date you set becomes the event date shown on the page, so set it carefully in the sidebar settings
  • Add Tag: event
  • Click Publish

7. How to Edit the Homepage (Section by Section)

a. The Hero Slider (Top Background Images)

  • Create ONE post and tag it hero-slider
  • Title: Anything you like (not shown on the site)
  • In the content, add several images one after another. These become the rotating full-screen backgrounds
  • The images rotate automatically, one every 5 seconds

b. The Two Focus Cards (Child Protection and Livelihood)

  • For the Child Protection card: create a post, write the title and short text, add a feature image, and tag it home-focus-child
  • For the Livelihood card: create a post, write the title and short text, add a feature image, and tag it home-focus-livelihood
  • Use only one post for each tag

c. The Focus Section Heading

  • Create ONE post and tag it home-focus-intro
  • Title: becomes the heading (e.g. "Two causes. One community.")
  • In the sidebar settings, add an Excerpt: becomes the subtitle paragraph

d. The "Our Story" Section

  • Create ONE post and tag it home-story
  • Title: becomes the heading
  • Body: type the story paragraphs

e. The Impact Stats (Numbers Band)

  • Create ONE post and tag it stats
  • In the content, add a bulleted list. For each stat, bold the number first, then a space, then the label
  • Example: 22+ Years of Service
  • To make the number bold, select it and press Ctrl+B
  • This same stats band shows on both the Home page and the About page
  • To update a number later, just open this post, change the number, and click Update

f. The Timeline (Milestones)

  • Create ONE post and tag it milestone
  • In the content, add a bulleted list. Bold the year first, then the description
  • Example: 2004 Founded by a group of women in Madhubani
  • Add a Feature Image: this photo shows beside the timeline on the homepage only (not on About)
  • This same timeline shows on both Home and About pages

g. The Partner Strip

  • Create ONE post and tag it partner
  • In the content, list each partner name as a bullet
  • The names scroll across the home page automatically

Each block on the About page is controlled by its own post. Create one post for each, with the tag shown below.

The post title becomes the heading and the content becomes the text. Use only one post per tag.

9. How to Edit the Donate Page

  • Create ONE post and tag it donate-info
  • Title: becomes the card heading (e.g. "For Indian Contributions")
  • Body: any intro text shown above the bank details (e.g. "Donate directly to our official bank account below.")
  • Excerpt (in sidebar settings): becomes the note below the bank details (e.g. "80G registered. Eligible for 50% tax deduction")

The bank account numbers themselves are set in Settings (see section 11 below).

10. How to Manage Pages

Pages are used for: About, Contact, Donate, and the section pages.

  • Click Pages in the left sidebar
  • Click on the page you want to edit
  • Make your changes
  • Click Update to save

Important: Each page has an Excerpt field (in sidebar settings). This is the description text shown below the page title on the website. Set it for each page.

12. How to Set Up Navigation (Header Menu)

The website menu is managed from Ghost Admin. Go to Settings, then Navigation.

Primary Navigation (top-level items):

LabelURL
Home/
Program#
Gallery#
Events/events/
Get Involved#
Donate/donate/

Items with URL # become dropdown menus. Items with a real URL become regular links. The "Donate" item becomes the red button on the right.

Secondary Navigation (dropdown sub-items):

LabelURL
Program: Ongoing Projects/ongoing-projects/
Program: Past Projects/past-projects/
Gallery: Photos/gallery/
Gallery: Videos/videos/
Get Involved: About Us/about/
Get Involved: Contact Us/contact/

The format is always: ParentLabel: ChildLabel (colon and space between parent and child). The parent name must match the primary nav label exactly.

Click Save. The footer links are also generated from this same navigation automatically.

13. How to Change Logo or Site Title

  • In the sidebar click Settings, then General, to change the Title and Description
  • For the Logo: click Settings, then Design and branding, then Customize
  • Upload the new logo (WebP format)
  • Click Save when done

14. The Language Button (English / Hindi)

The website has an EN and a Hindi button at the top right. Visitors click it to read the site in Hindi. It uses automatic translation and works on the live website. You do not need to maintain it.

16. Guidelines to Follow

  • Images must be WebP format only, do not upload JPG or PNG
  • Always add the correct Tag before publishing a post
  • Always add a Feature Image to ongoing project, gallery and video posts
  • For "one post" tags (partner, stats, milestone, hero-slider, home- and about- ones, donate-info), edit the existing post instead of making a new one
  • Do not delete any pages, only edit the content inside them
  • Do not change the page slugs (URLs) listed in section 10
  • Set the Excerpt on each page for the description text to appear
  • If you do not see your change, refresh the page with Ctrl + Shift + R together
  • Save before closing, always click Update or Publish before leaving
  • The Navigation menu (header and footer links) is managed from Settings, Navigation
